# MyHostex 智能回复助手 - Chrome 插件

自动读取 [MyHostex](https://www.myhostex.com/app/inbox) 收件箱新消息，提供智能回复建议，并持续学习你的回复风格。

## 功能特性

| 功能 | 说明 |
|------|------|
| 📬 消息监控 | 每5秒轮询收件箱，检测未读消息 |
| 💡 智能建议 | 根据消息内容意图，推荐合适的回复模板 |
| ↩ 一键发送 | 点击建议即可直接填充并发送回复 |
| 🤖 风格学习 | 记录用户的每次回复，分析语气、语言偏好、常用词汇 |
| 🎯 风格进化 | 学习样本越多，建议回复越像你自己 |
| 🔔 桌面通知 | 新消息到达时推送系统通知 |
| 📌 悬浮面板 | 不用时折叠到屏幕右侧，有新消息自动展开 |

## 安装方法

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 打开右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本目录 `myhostex-assistant/`
5. 插件安装完成，访问 https://www.myhostex.com/app/inbox 即可使用

## 使用说明

### 悬浮面板

- **收起状态**：显示为右侧紫色小图标，有新消息时图标会抖动并显示红色数字
- **展开状态**：显示完整面板，包含最新消息预览和建议回复
- **点击建议**：将建议文本填入自定义输入框（可进一步编辑）
- **点击发送按钮**：直接发送回复到当前对话

### 风格学习

插件会记录你发送的每一条回复，分析：
- **语言偏好**：中文 / 英文
- **语气风格**：正式 / 轻松
- **常用词汇**：高频出现的短语
- **平均长度**：你倾向于写多长的回复

学习数据越多，建议的回复越接近你的个人风格。

### 进化阶段

| 阶段 | 条数 | 说明 |
|------|------|------|
| 🥚 新手 | 0+ | 使用默认模板 |
| 🐣 萌芽 | 5+ | 开始融入语言偏好 |
| 🌱 成长 | 20+ | 掌握语气风格 |
| 🌿 进化 | 50+ | 注入常用表达 |
| ⭐ 熟练 | 100+ | 高度个性化建议 |
| 🏆 大师 | 200+ | 完全拟合你的风格 |

## 注意事项

- 插件仅在 `https://www.myhostex.com/` 域下运行
- 所有学习数据存储在本地 Chrome Storage，不上传服务器
- 如需清除学习数据，点击插件图标 → **清除学习数据**
