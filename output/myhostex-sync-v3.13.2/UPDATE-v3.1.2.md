# MyHostex 智能回复助手 v3.1.2 - 自然对话版本

## 🎯 更新目标

根据用户反馈，优化建议回复的风格，让回复更自然、更符合日常对话习惯。

## ✨ 主要改进

### 1. 优化 AI 系统提示词

**之前的风格**：
- 过于正式和客套
- 冗长的寒暄
- 机械化的模板回复

**现在的风格**：
- 自然、简洁，像日常聊天
- 直接回答问题，不需要太多寒暄
- 友好但不过于热情
- 接地气的表达方式

**系统提示词示例**：
```
你是一位经验丰富的民宿房东，擅长用自然、友好的方式与客人沟通。

重要原则：
1. 回复要自然、简洁，就像日常聊天一样，避免过于正式或客套
2. 直接回答问题，不需要太多寒暄
3. 语气要友好但不过于热情，保持专业的同时接地气
4. 根据消息内容生成 3-5 条不同风格的建议回复

回复风格示例：
- 客人说"你好" → "你好，有什么可以帮助您的？" 或 "您好，请问有什么需要吗？"
- 客人说"房间还有吗" → "有空的，请问您要住哪几天？" 或 "有的，您打算什么时候入住？"
- 客人说"多少钱" → "一天200元，您要住几天？" 或 "200一晚，住几天可以给您优惠"

避免：
- 过于正式的称呼（如"尊敬的客人"）
- 冗长的客套话
- 机械化的模板回复
- 过于热情的语气
```

### 2. 优化本地回退建议

**修改前（过于正式）**：
- 问候：`您好！感谢您的来信，请问有什么可以帮到您？`
- 价格：`感谢您的询价！价格详情请查看平台页面，如有特殊需求欢迎沟通。`
- 可用性：`您好！您查询的日期目前可预订，欢迎确认。`

**修改后（自然对话）**：
- 问候：`你好，有什么可以帮助您的？`、`您好，请问有什么需要吗？`
- 价格：`一天200元，您要住几天？`、`200一晚，住几天可以给您优惠`
- 可用性：`有空的，请问您要住哪几天？`、`有的，您打算什么时候入住？`

### 3. 优化 AI 参数配置

- **temperature**: 0.7 → 0.9（提高多样性，让回复更有变化）
- **maxTokens**: 500 → 200（缩短长度，保持简洁）

### 4. 增强关键词识别

新增常见对话关键词：
- `在吗` → 问候
- `多少钱一晚` → 价格询问
- `房租` → 价格询问
- `还能住吗`、`还能订吗` → 可用性询问
- `我要订`、`我要预订` → 确认预订
- `好的收到` → 感谢/确认

## 📋 回复对比示例

### 场景 1：客人打招呼

**之前**：
> 您好！感谢您的来信，请问有什么可以帮到您？
> Hi there! Thanks for reaching out. How can I help?

**现在**：
> 你好，有什么可以帮助您的？
> 您好，请问有什么需要吗？
> 你好呀，有什么我可以帮忙的？

### 场景 2：询问价格

**之前**：
> 感谢您的询价！价格详情请查看平台页面，如有特殊需求欢迎沟通。

**现在**：
> 一天200元，您要住几天？
> 200一晚，住几天可以给您优惠
> 房租200一天，您打算什么时候入住？

### 场景 3：询问是否有空房

**之前**：
> 您好！您查询的日期目前可预订，欢迎确认。

**现在**：
> 有空的，请问您要住哪几天？
> 有的，您打算什么时候入住？
> 可以预订，您要住几号到几号？

### 场景 4：确认预订

**之前**：
> 好的，已确认您的预订！期待您的到来，有任何问题随时联系。

**现在**：
> 好的，已为您预订成功
> 好的，收到，已确认
> 没问题，已经预订好了

### 场景 5：收到感谢

**之前**：
> 非常感谢您的好评！能让您满意是我们最大的开心。

**现在**：
> 不客气
> 没问题，应该的
> 好的，满意就好

## 🎨 自定义建议

如果你想进一步优化回复风格，可以在控制台执行：

```javascript
// 修改系统提示词
const newPrompt = `你是一位经验丰富的民宿房东。
回复风格：自然、简洁、接地气，就像日常聊天。
避免过于正式或客套。`;

chrome.storage.local.set({
  aiConfig: {
    ...await chrome.storage.local.get('aiConfig').then(r => r.aiConfig),
    systemPrompt: newPrompt
  }
});

// 修改温度参数
chrome.storage.local.set({
  aiConfig: {
    ...await chrome.storage.local.get('aiConfig').then(r => r.aiConfig),
    temperature: 0.95  // 更高温度，更随机
  }
});
```
