🔐 美团民宿 Token 提取工具

自动提取 x-phx-auth-token，用于 API 调用

📋 使用步骤

1
打开美团民宿房东后台并登录
https://minsu.dianping.com/host/house/list/online
2
点击下方按钮复制完整自动化脚本
3
在美团民宿页面按 F12 打开开发者工具
4
切换到 Console（控制台） 标签
5
粘贴脚本并运行，自动提取所有房源信息
💡 使用说明：
• 推荐使用增强版脚本 - 有更详细的日志和调试信息
• 脚本运行后，必须刷新页面（F5）才能触发数据捕获
• 如果刷新后没有数据，输入 showRequestLog() 查看拦截到的请求
• Token 可能有有效期限制，过期后需要重新提取
